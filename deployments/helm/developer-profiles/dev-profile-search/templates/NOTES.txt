=====================================================================
  QUICK INSTALL
=====================================================================

  helm upgrade --install vss-search ./dev-profile-search \
    -n default \
    --set global.externalHost=<NODE_IP> \
    --set global.ngcApiKey=<NGC_API_KEY> \
    --set nims.global.ngcApiKey=<NGC_API_KEY> \
    --set global.enableCritic=$ENABLE_CRITIC \
    --wait=false

  Ingress hosts are auto-constructed from global.externalHost using nip.io.
  Override with --set ingress.hosts.main=custom.domain.com if needed.
